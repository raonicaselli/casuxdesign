Design Process

  • User-centered Design

    This is the entire process I've been following in the last 2 years. It's based on the UCD methodology and it uses Problem Framing, Design Thinking and Lean UX for different stages of the process

    Read more

  • Problem framing

    First we have to understand really well the problem, who are the impacted actors, what are the difficulties around this problem, how is the current user experience and if it's a problem that really deserves to be solved

    Read more

  • Discovery Sprint with Design Thinking

    Design Thinking is a methodology that offers a solutions based problem solving approach. In a Discovery Sprint we can use it's methods and techniques to collect data, analyse it, generate insights and validate concepts for the next phase to work with.

    Read more

  • Lean UX

    Lean UX is all about understanding the problem and generating solutions to validate hypothesis. This is done by building MVPs (minimum viable prototypes) to test, getting feedback from real users to learn and evolve, in a cycle of continuous improvement and incremental design.
